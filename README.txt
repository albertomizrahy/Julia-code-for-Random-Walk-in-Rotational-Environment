README
There is 5 codes into this repository. 

1) RW_in_Rotational_Media.jl 

Julia code, with the main program.

2)Plot_1.jl

Julia code for the cross section plot.

3)Plot_2.jl

Julia code for the histogram plot. 

4)NeutronIntegra.jl

Julia code with the rate integral of the histogram x flux. 

5)Code.nb

Wolfram Mathematica code with the numerical solution of the burn/production chains. 

6)Data

Data folder used in the article, with cross section tables and Energy, inner product list.
